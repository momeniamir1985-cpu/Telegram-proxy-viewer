# Telegram Proxy Viewer 🚀

اسکریپت افزایش بازدید پست‌های تلگرام با استفاده از پروکسی‌های عمومی

## ویژگی‌ها
- جمع‌آوری خودکار پروکسی از منابع معتبر
- تست همزمان پروکسی‌ها با ThreadPool
- پشتیبانی از HTTP، SOCKS4 و SOCKS5
- حلقه بی‌نهایت برای افزایش مداوم بازدید

## نصب و اجرا در ترموکس
```bash
pkg update && pkg upgrade -y
pkg install python -y
pip install requests
python main.py
